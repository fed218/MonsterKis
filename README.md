## 簡介
* 前台：
    - 商品展示 ( 電影資訊 / 影城餐飲 )
    - 下單 / 結帳
    - 購物車
    - 優惠券套用
* 後台：
    - 商品管理 ( 新增 / 修改 / 刪除 )
    - 訂單管理
    - 優惠券管理 ( 新增 / 修改 / 刪除 )

## 主要使用技術
* Vue Cli 3.0 
* Vue Router 
* Vuex
* RESTful API 串接
* RWD
* Bootstrap 4
* SCSS

## Plugin

* vee-validate
* Animate.css
